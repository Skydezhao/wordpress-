# wordpress-
简单的wordpress网站文章发布小工具
使用前请在相同目录下创建一个article文件夹
可一次性批量上传多个文件，目前仅支持.docx文件，程序中留有.doc转化.docx的函数，可自行调用
注意使用前请配置自己的网站地址，用户名，密码以及上传模式
